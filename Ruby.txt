Installation de Rails
---------------------------------------------------------
-> http://railsinstaller.org/fr-FR

Installation des bundles
---------------------------------------------------------
c:/>gem install bundler

Redémarrage commande

Création du projet
---------------------------------------------------------
c:/chemin/dossier/creation/projet>rails new <nom projet>

Edition fichier Gemfile
---------------------------------------------------------
Décommenter => gem 'therubyracer'

Installation de toutes les gem necessaire au projet présente dans le Gemfile
---------------------------------------------------------
c:/chemin/dossier/creation/projet>bundle install

Lancer serveur ruby
---------------------------------------------------------
c:/chemin/projet>bin/rails server

Navigateur
---------------------------------------------------------
http://localhost:3000

Ajout de mysql
---------------------------------------------------------
Ajouter en bas du fichier Gemfile => gem mysql2
c:/chemin/projet>bundle install

Modifier le fichier /config/database.yml :

# SQLite version 3.x
#   gem install sqlite3
#
#   Ensure the SQLite 3 gem is defined in your Gemfile
#   gem 'sqlite3'
#
default: &default
  adapter: mysql2
  pool: 5
  timeout: 5000

development:
  adapter: mysql2
  database: pokemons
  username: root
  password:
  host: localhost

# Warning: The database defined as "test" will be erased and
# re-generated from your development database when you run "rake".
# Do not set this db to the same as development or production.
test:
  <<: *default
  database: db/test.mysql2

production:
  <<: *default
  database: db/production.mysql2

Faire attention à l'indentation !!!

Génération de migration
---------------------------------------------------------
c:/chemin/projet>bin/rails generate migration CreateArticle name:string

Création base
---------------------------------------------------------
c:/chemin/projet>bin/rake db:create

Faire migration
---------------------------------------------------------
c:/chemin/projet>bin/rake db:migrate
